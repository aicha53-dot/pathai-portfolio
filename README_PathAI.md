# PathAI — Livrable & Cahier des charges

Ce dossier contient une page HTML autonome pour présenter le projet PathAI comme un livrable professionnel/portfolio.

## Fichiers
- `pathai_livrable_cahier_des_charges.html` : page principale prête à ouvrir dans le navigateur.

## Mise en ligne GitHub Pages
1. Créer un repository GitHub.
2. Renommer le fichier HTML en `index.html`.
3. Ajouter le fichier dans le repository.
4. Aller dans `Settings > Pages` puis choisir la branche `main`.
5. Le site sera publié automatiquement.

## Déploiement Vercel / Netlify
Déposer le fichier `index.html` dans un dossier et l’importer dans Vercel ou Netlify.
