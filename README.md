# PathAI — Livrable HTML

Ce dossier contient une version HTML premium du **livrable + cahier des charges** du projet **PathAI**.

## Fichiers
- `index.html` : page principale
- `pathai_assets/` : images extraites du PDF pour enrichir le design

## Mise en ligne sur GitHub Pages
1. Créer un nouveau repository GitHub.
2. Uploader `index.html` et le dossier `pathai_assets`.
3. Aller dans **Settings** > **Pages**.
4. Dans **Build and deployment**, choisir **Deploy from a branch**.
5. Choisir la branche `main` et le dossier `/root`.
6. Cliquer sur **Save**.
7. Attendre 1 à 3 minutes.
8. Ouvrir le lien GitHub Pages généré.
